# Magic the Gathering Cards More Expensive than an iPhone Pro Max

A simple web app for displaying 6 random mtg cards from the scryfall API that are over $1000 USD

## Technologies USED

- HTML5
- CSS3
- JavaScript/jQuery
- scryfall API
